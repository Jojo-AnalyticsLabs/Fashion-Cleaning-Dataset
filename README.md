# Fashion Dataset Cleaning Project

## Overview

This project focuses on cleaning and preparing a fashion dataset using Python and Pandas. The dataset contained missing values, duplicate records, inconsistent formatting, and unnecessary columns that were cleaned to improve data quality and prepare the data for analysis.

## Objectives

* Improve dataset quality
* Removing Redundant columns
* Handle missing values
* Remove duplicate records
* Transform Nan Values from the dataset
* Standardize data formatting
* Prepare clean data for analysis and visualization

## Skills Demonstrated

* Data Cleaning
* Handling Missing Values (Nan)
* Removing Duplicates
* Renaming Columns
* Data Transformation
* Exporting Cleaned Data
* Exploratory Data Preparation

## Tools & Technologies

* Python
* Pandas
* Jupyter Notebook
* VS Code

## Cleaning Process

The following cleaning tasks were performed:

* Removed unnecessary columns
* Handled missing values in selected columns
* Checked and removed duplicate rows
* Standardized column names
* Cleaned text values
* Exported the cleaned dataset for further analysis

## Files Included

* Raw Fashion Dataset
* Cleaned Fashion Dataset
* Jupyter Notebook (.ipynb)
* README.md

## Outcome

The cleaned dataset is now more structured, consistent, and ready for data analysis, visualization, and future machine learning projects.
